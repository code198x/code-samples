# ZX Spectrum Lesson 007 - First Pixels

## Files

- `plot-demo.bas` - Basic PLOT demonstration with crosshair
- `pixel-patterns.bas` - Box and diagonal patterns

## Key Concepts

- PLOT x,y draws pixels (x: 0-255, y: 0-175)
- Y coordinates inverted (0=bottom, 175=top)
- Use FOR loops to draw lines and shapes
